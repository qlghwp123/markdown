# markdown

my own summary of markdown


## 1. Header 

To make header, try ```#```.  

```
# h1
## h2
### h3
#### h4
##### h5
###### h6
```

# h1

## h2

### h3

#### h4

##### h5

###### h6


## 2. List

to use unordered list, try one of these ```- * +```.

```
- '-' use
- '-' use

* '*' use
* '*' use

+ '+' use
+ '+' use
```


- '-' use
- '-' use

* '*' use
* '*' use

+ '+' use
+ '+' use

to use ordered list, start with ```1.  ```

```
1. first
2. second
3. third
```

1. first
2. second
3. third


## 3. Blockquotes

to make Blockquote, try ```>``` and also try ```>>``` if u want to make Blockquote inside Blockquote.

> first
>
> > second
